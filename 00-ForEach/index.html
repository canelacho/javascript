forEach() - Javascript

'strict' : A good practice to not use undeclared variables.

'strict'

We create an Array of animals

animals = ['Dog','Cat','Mouse','Bird']

Now we apply the forEach() method to the animals array.

animals.forEach( animal => console.log(animal) )

Each value of the array is loaded in the "animal" argument.
Each "animal" will be show in the console.

    Explanation

  • Firs we create a new Array with values that represent animals: const animals = ['Dog','Cat','Mouse','Bird']
  • Resolving the `forEach()` method applied to our "`animals`" array.
  • The `forEach()` method take the "`animals`" array and evaluate each value.
  • Each iteration is processed one by one.
  • Using arrow function we pass "`animal`" as an argument to show in console the type of each animal on the array.
  • When the `forEach()` method finish to evaluate all the positions in the array, it finish.
